openAIDA GPTs API Privacy Policy

Effective Date: January 1, 2025

Introduction

Welcome to the openAIDA GPTs API. This privacy policy applies to all users of the openAIDA GPTs API. We are committed to protecting your privacy.

Note: The GPTs API does not handle personal information.

Data Handling Practices

Authentication

The GPTs API does not store any personal data from the GPTs or the authentication process.

Data Collection

We collect interaction data such as:

  • API usage patterns
  • Request types
  • Response data

to improve service performance and enhance user experience.

Data Storage

All collected data is securely stored, focusing on non-identifiable data only.

Content Generation and Rights

Content Creation

The openAIDA GPTs API allows for dynamic content generation based on user requests.

Anonymity and Access

Generated content is anonymized and cannot be attributed to any individual user.

Rights and Distribution

We retain the rights to use, distribute, and repurpose the generated content.

Data Sharing

Requests may involve communication with external services for processing and improvement purposes, limited to non-identifiable interaction data.

Policy Modifications

We reserve the right to modify this privacy policy to reflect changes in our practices and regulations.
